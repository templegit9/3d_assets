## Common Issues

### "Python not found"
Install Python 3.10 from python.org

### "CUDA error" or "GPU not available"
Update NVIDIA drivers from nvidia.com

### "Compilation failed"
Install Visual C++ Build Tools (link above)

### "Out of memory"
Make sure you're using `run.bat` (has low VRAM mode enabled)
